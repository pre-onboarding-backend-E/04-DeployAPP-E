4주차 과제 / Amazon EC2를 이용하여, Free-Tier 서버를 생성합니다.

1. AWS EC2 인스턴스 생성 및 배포
2. EC2 내에서 Nestjs 프로젝트 실행
3. Elastic IP 적용
4. return Hello
